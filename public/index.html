API

Para iniciar a aplicação:

npm install
npm run dev

Para rodar os poucos testes que eu fiz:

npm test

Endpoints

MÉTODO ENDPOINTS DESCRIÇÃO
GET /appointments Listar todas as regras
GET /appointments?startDate=DD-MM-YYYY&endDate=DD-MM-YYYY
 Listar dentro de um intervalo
POST /appointments
 Criar regra
DELETE /appointments/:id Remover regra
